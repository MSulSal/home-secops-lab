# Lab 07 - Anomaly Detection

## Goal

Implement explainable anomaly detection over network telemetry.

## Techniques

- Periodicity detection for beaconing-like behavior
- EWMA-based deviation scoring
- Isolation Forest over engineered features

## Tasks

1. Build parsers for Zeek, DNS, and Suricata exports.
2. Engineer session and domain-level features.
3. Generate anomaly report with ranked findings and rationale.

## Validation

Produce a human-readable report and analyst follow-up recommendations.
