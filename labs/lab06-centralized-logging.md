# Lab 06 - Centralized Logging

## Goal

Deploy Loki + Promtail pipeline and integrate Grafana dashboards.

## Tasks

1. Configure source labels and log routing.
2. Build Grafana dashboards for DNS, Zeek, and Suricata views.
3. Validate retention and query performance.

## Validation

Confirm end-to-end visibility across all telemetry sources.
