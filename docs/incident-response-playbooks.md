# Incident Response Playbooks

## Purpose

Provide repeatable defensive response procedures for common detection scenarios.

## Core Playbooks

- Suspicious DNS beaconing investigation
- High-severity IDS alert triage
- Newly observed external destination with elevated risk

## Playbook Template

1. Trigger condition
2. Triage data sources
3. Containment steps
4. Eradication and recovery checks
5. Post-incident hardening actions
